Arrancar el proyecto
====================

Instrucciones rápidas para iniciar y depurar el proyecto desde VS Code.

- Crea un archivo `.env` en la raíz del proyecto a partir de [/.env.example](.env.example).
- Sigue los pasos detallados en [README_ENV.md](README_ENV.md) para configurar `launch.json` (usando el menú Run and Debug) y añadir `envFile`.
